## 三、网站部署

### 当前部署方式

**平台**: Vercel（免费）

**部署流程**:
```
GitHub仓库 → Vercel自动拉取 → 构建部署 → 全球CDN
```

**仓库地址**: https://github.com/XX1483732026/-.git

**自动部署规则**:
- main分支推送 → 自动触发部署
- 部署时间: 约1-2分钟
- 部署日志: Vercel控制台可查

**Vercel配置**:
```json
{
  "buildCommand": null,
  "outputDirectory": "./",
  "framework": null,
  "rewrites": [
    { "source": "/api/(.*)", "destination": "/api/$1" }
  ]
}
```

### 迁移后部署方式

**平台**: 腾讯云服务器

**部署架构**:
```
Nginx（反向代理 + SSL）
    │
    ├── 静态资源：/var/www/xingxingya/
    │
    └── API服务：PM2管理Node.js进程
                  │
                  └── 监听 localhost:3000
```

**部署流程**:
1. 宝塔面板安装 Nginx + Node.js
2. Git clone 代码到服务器
3. 配置 Nginx 反向代理
4. PM2 启动 API 服务
5. 配置 SSL 证书（Let's Encrypt）

- 22页HTML阅读器

## 七、数据流

### 聊天流程
```
用户输入 + 文件
    ↓
chat.html（前端）
    ↓ 文件转Base64
api/chat.js（Vercel Functions）
    ↓ 解析文件：Word/PDF/TXT → 纯文本
    ↓ 组装消息
Coze API（小鸦智能体）
    ↓ 流式返回（SSE）
api/chat.js → 逐字返回
    ↓
chat.html → 渲染回复
```

### 测试流程
```
用户答题（15题）
    ↓
test.html 计算得分
    ↓ 4个维度：电量/情绪/行动/连接
匹配鸦格状态
    ↓
显示结果 + 可分享
```
