## 四、网站部署

### 当前部署方式（Vercel）

**平台**: Vercel（免费）

**流程**:
```
GitHub Push → Vercel监听 → 自动拉取代码 → 构建 → 全球CDN分发
```

**优势**:
- 自动部署，无需手动操作
- 全球CDN加速
- 免费SSL证书
- 部署日志可查

**限制**:
- API函数有执行时间限制
- 无本地文件存储
- 国内访问有时较慢

### 迁移后部署方式（腾讯云）

**平台**: 腾讯云服务器

**流程**:
```
首次部署：git clone 到服务器
更新代码：git pull + 重启服务
```

**架构**:
```
Nginx（80/443端口）
    │
    ├── 静态文件: /var/www/xingxingya/
    │
    └── 反向代理: /api/* → localhost:3000
                         │
                    PM2管理Node.js进程
```

**优势**:
- 完全自主控制
- 支持文件存储
- 访问统计持久化
- 国内访问更快

## 六、核心功能

### 1. 鸦格测试
- 12种鸦格状态（清醒区/假寐区/沉睡区）
- 15道题目评估

### 2. 小鸦聊天
- 流式对话（SSE）
- 图片上传（Base64编码）
- 文件上传（Word/PDF/TXT → 后端解析转文本）

### 3. 访问统计
- 前端埋点采集

- 密码保护查看页

### 4. 绘本功能
- 从文本/图片生成交互式绘本
- 22页HTML阅读器
- PDF导出

## 七、数据流

### 聊天流程
```
用户输入 + 文件
    ↓
chat.html（前端）
    ↓ 文件转Base64
api/chat.js（Vercel Functions）
    ↓ 解析文件：Word/PDF/TXT → 纯文本
    ↓ 组装消息
Coze API（小鸦智能体）
    ↓ 流式返回（SSE）
api/chat.js → 逐字返回
    ↓
chat.html → 渲染回复
```
