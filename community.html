什么是鸦格？

鸦格是醒醒鸦的12种心境状态，通过4个维度（电量、情绪、行动、连接）
计算出你此刻的精神状态。不是标签，是当下真实的你。

🌅 清醒区
电量充沛，状态在线，掌控感满满
🌅 清醒区
🦅
巡天智者鸦
电量高 + 情绪高/中 + 连接高/中
此刻的你，像一只盘旋在高空中的鸦，看得见地面的拥堵，也看得见远方的旷野。你没有被忙碌淹没，也没有被焦虑吞噬。
🐦‍⬛ 鸦鸦有话说
清醒不是冷漠，是看透后依然选择认真生活。
🎯 适合：做重要决策、处理复杂问题、深度交流
🌅 清醒区
🦅
待机隐士鸦
电量高 + 情绪低
你现在的能量满格，但选择了蛰伏。像一只在枝头安静待着的鸦，目光深远，随时可以展翅，但没有那个冲动。
🐦‍⬛ 鸦鸦有话说
待机不是懈怠，是在等一个值得全力以赴的时刻。
🎯 适合：观察思考、等待时机、深度工作
🌅 清醒区
🔥
行动爆发鸦
电量高 + 行动高
你现在像一团刚被点燃的火，噼里啪啦地烧着。别人还在犹豫，你已经冲出去了；别人还在计划，你已经完成了一半。
🐦‍⬛ 鸦鸦有话说
热情是好燃料，但记得系好安全带再踩油门。
🎯 适合：执行计划、创意输出、社交活跃
🌫️ 假寐区
电量平稳，不上不下，一切刚刚好
🌫️ 假寐区
🐟
舒适咸鱼鸦
电量中 + 情绪高 + 行动低
一切都刚刚好：不太累，不太饿，不太冷，不太热。就是……不太想动。这不是颓废，这是对自己辛勤工作的奖励。
🐦‍⬛ 鸦鸦有话说
咸鱼也需要资格认证，你刚好是那个持证上岗的人。
🎯 适合：休息恢复、低强度娱乐、发呆放空
🌫️ 假寐区
🦜
社交探索鸦
电量中 + 情绪高 + 行动高
电量不算特别足，但心情好啊！脑子里冒出一堆想去的地方、想见的人、想做的事。今天你就是那只最闪亮的社交鸦。
🐦‍⬛ 鸦鸦有话说
好心情是最好的燃料，一起出发才更带劲。
🎯 适合：组队活动、短途出行、社交聚会
🌫️ 假寐区
🚉
电量中转鸦
电量中 + 情绪中 + 行动高
你现在处于一个「正在转换」的节点。像站在中转站的鸦，等待自己的下一班列车。不用急着做决定，在这里停一停。
🐦‍⬛ 鸦鸦有话说
中转站不是终点，是蓄力的好时候。
🎯 适合：整理思路、处理杂事、等待时机
🌫️ 假寐区
🌫️
迷雾探险鸦
电量中 + 情绪低/中 + 行动中 + 连接低/中
前方的路看不清，但你还是迈出了步子。迷茫说明你还在寻找，还没有放弃。迷雾会散的，路会明的。
🐦‍⬛ 鸦鸦有话说
看不清路的时候，走本身就是答案。
🎯 适合：小范围试错、收集信息、聊天交流
🌫️ 假寐区
🎭
伪装讨喜鸦
电量中/低 + 情绪低 + 连接高
你每天戴着「我很好」的面具，在人群里周旋。别人都觉得你游刃有余，只有你自己知道，摘下面具的那一刻有多疲惫。
🐦‍⬛ 鸦鸦有话说
你不需要对所有人笑，有些人值得看你真实的眼泪。
🎯 适合：维持基本社交、必要应酬
🌑 沉睡区
电量告急，需要休息和恢复
🌑 沉睡区
🌙
疗愈独处鸦
电量低 + 情绪高 + 行动中/低
电量快见底了，但心情居然还不错。像大雨后的晴天，有一点疲惫，但更多的是宁静和满足。这种自我疗愈的能力，是成年人的必修课。
🐦‍⬛ 鸦鸦有话说
最好的疗愈，有时候就是什么都不做。
🎯 适合：安静休息、简单娱乐、独自恢复
🌑 沉睡区
⚡
电量透支鸦
电量低 + 情绪高 + 行动高
明明已经累到不行了，但心情却很亢奋。像一辆没油的车还在跑，靠的是最后的意志力。冷静下来，让身体休息。
🐦‍⬛ 鸦鸦有话说
亢奋是最后的电量，且用且珍惜。
🎯 适合：紧急任务（慎用！）、必要坚持
🌑 沉睡区
💨
燃尽冲刺鸦
电量低 + 情绪低 + 行动高
已经很累了，但停不下来。可能是 deadline，可能是承诺没兑现，可能是单纯的不甘心。完成比完美重要。
🐦‍⬛ 鸦鸦有话说
冲刺很好，但记得在耗尽前给自己留点余量。
🎯 适合：意志力对抗（慎用！）
🌑 沉睡区
🌅
破晓微光鸦
电量低 + 情绪低 + 行动中/低
你刚刚经历了一些不太好的事，可能还没完全走出来。但你今天愿意面对自己，这本身就是一件很勇敢的事。
🐦‍⬛ 鸦鸦有话说
最低谷的时候，往哪个方向走都是向上。
🎯 适合：低强度任务、陪伴治愈、自我觉察
